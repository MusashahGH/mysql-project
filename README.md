# 🗄️ SQL Practice (MySQL)

SQL practice queries written in MySQL as part of my DBMS coursework.

---

## 📌 Project Description

This repository contains **SQL practice queries** demonstrating fundamental database operations using MySQL. The project includes a complete **Library Database** with a `books` table and various queries showcasing different SQL concepts.

---

## 📚 Projects Included

### 📖 Library Database
A library management database with a `books` table containing:
- 📊 Book ID (Primary Key)
- 📝 Title, Author, Genre
- 💰 Price, Copies, Availability
